## Integration

Value must be a date string following the [ISO 8601 datetime format standard](https://www.w3.org/TR/NOTE-datetime),
`1996-12-19`. The format does not have to be specific to an exact datetime.
For example, the maximum could just be the year, such as `1994`. Defaults to the end of this year.

```typescript
import { formatDateString, now } from '@baloise/web-app-utils'

const datepickerDisabled = document.getElementById('datepicker-disabled')
datepickerDisabled.value = formatDateString(now())
// or
datepickerDisabled.value = formatDateString(2020, 0, 13)
// or
datepickerDisabled.value = formatDateString(new Date(2020, 0, 13))
```
