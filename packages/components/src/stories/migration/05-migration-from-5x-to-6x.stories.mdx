# Migration from 5.x to 6.x

The following workflow walks you through the migration steps.
Note that the actual steps required for your project may vary, and these steps should be treated as general guidance rather than strict instructions.

## CSS helper classes

- replace `has-border-radius` with `has-radius-normal`
- replace `has-border-blue-dark` with `has-border-dark`
- replace `has-border-blue-light` with `has-border-light`
- replace `has-no-border` with `has-border-none`
