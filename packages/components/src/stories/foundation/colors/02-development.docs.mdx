## Backgrounds

Background colors can be set with the CSS class `has-background-***` for example `has-background-green`.

Every color of the [Design Tokens](?path=/docs/foundation-colors-design-tokens--page) can be used like that.

<Canvas withSource="open">
  <Story id="foundation-colors-development--background-color" />
</Canvas>

## Typography

The classes `has-text-xxx` defines the font-color.
For more colors have a look a the [Font Color Tokens](?path=/docs/foundation-typography-design-tokens--page#colors).

<Canvas withSource="open">
  <Story id="foundation-colors-development--text-colors" />
</Canvas>

## Borders

The classes `has-border-xxx` defines the border-color and the width.
For more colors have a look a the [Border Color Tokens](?path=/docs/foundation-border-radius-design-tokens--page#colors).

## Installation

To install the CSS Framework run the following npm command:

```
npm add @baloise/design-system-css
```

> **TIP**
>
> The CSS package is already included in the **component** packages.
> To use the CSS Utility Classes import the following stylesheet.

```scss
@import '@baloise/design-system-css/css/colors';
```
