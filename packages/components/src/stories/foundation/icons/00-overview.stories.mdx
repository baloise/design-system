## Overview

We provide up to 100 vector graphics of the collection [Core Solid Pro](https://www.streamlinehq.com/icons/streamline-mini-bold)
of [Streamline](https://www.streamlinehq.com)

These are the icons we deliver in the package `@baloise/design-system-icons`.

If there is one missing please feel free to create a [GitHub Issue](https://github.com/baloise/design-system/issues/new?assignees=Gagne87%2C+clastzoo&labels=%F0%9F%8E%A8+Design+Support&template=design-request.md).
