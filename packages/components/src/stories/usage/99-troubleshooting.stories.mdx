# Troubleshooting

## Bundle size warning

**WARNING in budgets, maximum exceeded for initial**

Open angular.json file and find budgets keyword and adjust the two values.

```json
"budgets": [
  {
    "type": "initial",
    "maximumWarning": "3mb",
    "maximumError": "5mb"
  }
]
```
