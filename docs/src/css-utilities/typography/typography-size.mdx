<Lead>Specifies the font size of an element.</Lead>

## Classes

The font-size CSS utility classes are inherently **responsive**. This means that the size automatically adjusts itself based on breakpoints such as mobile, tablet, and desktop.

<CssTypographySize />

## Headings

The default size for the `title` class is `text-xxx-large`, which corresponds to the size of a heading 1 element.

<CanvasWithCodePen of={TypographySizeStories.Headings} />

## Texts

The default size for the `p` element is `text-normal`.

<CanvasWithCodePen of={TypographySizeStories.Texts} />

## Display

The two display sizes are designed for large-scale website layouts.

<CanvasWithCodePen of={TypographySizeStories.Display} />

## Responsive

Tailored responsive options are at your disposal for customizations according to screen size.
Simply append a responsive breakpoint keyword with a dash, like `tablet:text-large`, to apply the corresponding responsive class.
