<Lead>A shorthand property for setting `flex-grow`, `flex-shrink`, and `flex-basis` simultaneously.</Lead>

## Classes

<CssFlexboxFlex />

## Initial

The flex-initial setting configures `flex-grow` to 0, `flex-shrink` to 1, and `flex-basis` to auto, which is the default browser behavior.

<CanvasWithCodePen of={FlexboxFlexStories.Initial} />

## Flex 1

Establishes both `grow` and `shrink` factors to 1, enabling automatic expansion and contraction, while maintaining a `flex-basis` of 0.

<CanvasWithCodePen of={FlexboxFlexStories.Flex1} />

## Auto

By setting both `grow` and `shrink` factors to 1, the item will automatically expand and contract, while maintaining a `flex-basis` of auto.

<CanvasWithCodePen of={FlexboxFlexStories.Auto} />

## None

Equivalent to `0 0 auto`, indicating that the element neither expands nor contracts.

<CanvasWithCodePen of={FlexboxFlexStories.None} />

## Responsive

Tailored responsive options are at your disposal for customizations according to screen size.
Simply append a responsive breakpoint keyword with a dash, like `tablet:flex-1`, to apply the corresponding responsive class.
