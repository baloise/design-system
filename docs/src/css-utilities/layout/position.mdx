## Static

An element with a static position is rendered in accordance with the natural flow of the page and remains unaffected by adjustments made through the top, left, right, and bottom properties.

<CanvasWithCodePen of={PositionStories.Static} />

## Fixed

A fixed-positioned element is situated with respect to the viewport and maintains a constant position, unaffected by scrolling movements within the page.

<CanvasWithCodePen of={PositionStories.Fixed} />

## Relative

An element with a relative position is presented relative to its initial placement. Utilizing the top, left, right, and bottom properties allows for adjustments to the element's position.

<CanvasWithCodePen of={PositionStories.Relative} />

## Absolute

A fixed element is placed in relation to the viewport and remains consistently positioned, unaffected by scrolling actions on the page.

<CanvasWithCodePen of={PositionStories.Absolute} />

## Sticky

A sticky element dynamically adjusts its position according to the scroll position, transitioning between relative and fixed behaviors.

<CanvasWithCodePen of={PositionStories.Sticky} />

## Responsive

Tailored responsive options are at your disposal for customizations according to screen size.
Simply append a responsive breakpoint keyword with a dash, like `tablet:absolute`, to apply the corresponding responsive class.
