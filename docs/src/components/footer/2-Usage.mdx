## When to Use

- Display application-level legal links, copyright information, and contact details
- Provide language selection for multilingual applications
- Include social media links and branding
- Present navigation options for privacy policies and imprints

## When NOT to Use

- For inline content that doesn't belong at the page bottom
- As a replacement for main navigation (use header components instead)
- For temporary or contextual information (use alerts or modals)
