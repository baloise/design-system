## WCAG 2.2 AA Compliance

The footer component implements patterns that meet WCAG 2.2 Level AA standards:

- **Link Purpose (2.4.9)** — All links have descriptive text or aria-labels
- **Language of Page (3.1.1)** — Language selection allows users to choose preferred language
- **Keyboard Navigation (2.1.1)** — All interactive elements are keyboard accessible
- **Color Contrast (1.4.3)** — Text and links meet minimum contrast ratios
- **Focus Visible (2.4.7)** — Keyboard focus indicators are clearly visible
