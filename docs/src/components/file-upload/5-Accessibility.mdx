## WCAG 2.2 AA Compliance

The file-upload component implements patterns that meet WCAG 2.2 Level AA standards:

- **Form Labels (1.3.1)** — File uploads are associated with labels via the `label` prop
- **Instructions (3.3.2)** — The `description` prop provides context and requirements
- **Error Identification (3.3.1)** — Validation errors are announced via `invalidText` and visual indicators
- **Keyboard Access (2.1.1)** — All functionality is accessible via keyboard navigation
- **Focus Management (2.4.7)** — Focus is visible and properly managed during interactions
- **Color Contrast (1.4.11)** — Text colors meet minimum contrast ratios against backgrounds
