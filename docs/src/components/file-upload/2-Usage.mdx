## When to Use

- Allow users to upload single or multiple files via drag-and-drop
- Provide file validation based on type, size, or quantity
- Integrate file upload into forms with optional validation states
- Display a list of selected files with the ability to remove them
- Show upload progress or loading state during file processing

## When NOT to Use

- For simple button-based file selection without drag-and-drop interaction
- When multiple independent uploads need separate handling
- For file management interfaces that require advanced features like preview, reordering, or editing
