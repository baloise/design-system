## WCAG 2.2 AA Compliance

The radio component supports patterns that meet WCAG 2.2 Level AA standards:

- **Grouped Inputs (1.3.1)** — Radio groups use fieldset/legend
- **Labels (1.3.1)** — Radio buttons have associated labels
- **Keyboard Navigation (2.1.1)** — Arrow keys navigate options, Tab moves between groups
- **Focus Visible (2.4.7)** — Clear focus indicator on radio buttons
- **Name, Role, Value (4.1.2)** — Proper semantics and ARIA support
