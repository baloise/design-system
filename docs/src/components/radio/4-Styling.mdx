## CSS Shadow Parts

Shadow DOM parts are named elements within a web component that can be targeted with CSS selectors from outside the component, allowing you to style specific internal elements without breaking encapsulation.

<br />

**Example:** Target the radio button:

```css
/* Style the radio element */
ds-radio::part(control) {
  width: 20px;
  height: 20px;
  border-radius: 50%;
}
```

<br />

Primary parts of the radio component include:

| Part | Element | Purpose |
| --- | --- | --- |
| `control` | Div | The radio button visual indicator. |

## Design Tokens

CSS variables (custom properties) allow you to define reusable values that can be customized globally or per component instance to override default styling without modifying the component code.

<br />

**Example:** Override radio styling:

```css
/* Override globally */
:root {
  --ds-radio-color-background: #ffffff;
  --ds-radio-color-border: #cccccc;
}

/* Override for a specific radio */
ds-radio {
  --radio-color-checked: #005eb8;
}
```
