## When to Use

- Select one option from a small set of mutually exclusive choices (2-5 options)
- Choose a single option that affects the entire form or state
- Select from predefined categories or types
- Make a yes/no decision in a clear, visual way

## When NOT to Use

- For multiple selections — use checkbox component instead
- When there are more than 5-6 options — use select dropdown instead
- For filtering that allows multiple selections — use checkboxes
- When space is limited and options are many — consider a select dropdown
- For dynamic or searchable lists — use appropriate input or select component

## Do's and Don'ts

### Radio Group Labels

Always provide a group label and individual option labels. Never rely on values alone.
