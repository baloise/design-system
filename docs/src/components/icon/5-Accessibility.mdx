## WCAG 2.2 AA Compliance

The icon component supports patterns that meet WCAG 2.2 Level AA standards:

- **Text Alternatives (1.1.1)** — Icons have text labels or ARIA labels
- **Color Contrast (4.11.1)** — Icons have sufficient contrast against backgrounds
- **Name, Role, Value (4.1.2)** — Proper ARIA labels on icon-only interactive elements
- **Visual Size (2.5.5)** — Icons are large enough to be easily clickable
