## WCAG 2.2 AA Compliance

The number-input component supports patterns that meet WCAG 2.2 Level AA standards:

- **Labels (1.3.1)** — Inputs have associated labels
- **Identification of Input (3.3.2)** — Input purpose is clearly labeled
- **Focus Visible (2.4.7)** — Clear focus indicator
- **Keyboard Accessible (2.1.1)** — Arrow keys and buttons are keyboard accessible
- **Name, Role, Value (4.1.2)** — Proper semantics for buttons and input
