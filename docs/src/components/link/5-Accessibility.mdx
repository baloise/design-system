## WCAG 2.2 AA Compliance

The link component supports patterns that meet WCAG 2.2 Level AA standards:

- **Meaningful Text (1.1.1)** — Descriptive link text explains destination
- **Semantic Markup (1.3.1)** — Uses proper `<a>` element
- **Link Purpose (2.4.9)** — Link text explains purpose or destination
- **Distinguishable (1.4.3)** — Links visually distinguished from surrounding text
- **Focus Visible (2.4.7)** — Keyboard focus indicator visible
- **Color Contrast (1.4.3)** — Sufficient contrast for link colors
