## When to Use

- Navigate to other pages or sections
- Link to external resources or documentation
- Create inline navigation within content
- Build navigation menus and breadcrumbs
- Provide "Learn More" or call-to-action links
- Link to related resources

## When NOT to Use

- For buttons that trigger actions — use button component
- For anchor links without navigation — use standard HTML
- For social media sharing — use specific social components
- For download links — consider button or download component
- For email mailto links — use semantic HTML mailto

## Do's and Don'ts

### Descriptive Link Text

Always use meaningful link text. Avoid generic "Click here" or "Read more".
