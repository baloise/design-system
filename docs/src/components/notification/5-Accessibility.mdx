## WCAG 2.2 AA Compliance

The notification component supports patterns that meet WCAG 2.2 Level AA standards:

- **Color Contrast (4.11.1)** — Text meets minimum contrast ratios
- **Color Not Only Means (1.4.1)** — Color is not the only means of conveying information
- **Status Messages (4.1.3)** — Live region announcements for dynamic content
- **Focus Visible (2.4.7)** — Close buttons have visible focus indicators
