## Design Tokens

CSS variables (custom properties) allow you to define reusable values that can be customized globally or per component instance to override default styling without modifying the component code.

<br />

**Example:** Override button colors using CSS variables:

```css
/* Override globally for all primary buttons */
:root {
  --ds-button-color-primary-base-text: #ffffff;
  --ds-button-color-primary-base-background: #005eb8;
}

/* Override for a specific button instance */
ds-button {
  --button-color-background: #fae052;
  --button-color-text: #1a1a1a;
}
```
