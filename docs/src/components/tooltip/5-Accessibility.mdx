## WCAG 2.2 AA Compliance

The tooltip component implements patterns that meet WCAG 2.2 Level AA standards:

- **1.3.1 Info and Relationships** — The trigger element receives `aria-describedby` pointing to the tooltip `id`, making the relationship machine-readable.
- **1.4.13 Content on Hover or Focus** — The tooltip persists while the pointer is over the trigger and can be dismissed with Escape; it does not disappear on hover alone.
- **2.1.1 Keyboard** — The tooltip is triggered on both `mouseenter` and `focus`, so it is accessible without a pointing device.
- **4.1.3 Status Messages** — The tooltip uses `role="tooltip"` and `aria-hidden` to correctly communicate its visibility state to assistive technology without moving focus.
