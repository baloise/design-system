## WCAG 2.2 AA Compliance

The input component supports patterns that meet WCAG 2.2 Level AA standards:

- **Labels (1.3.1)** — Inputs have properly associated labels
- **Identification of Input (3.3.2)** — Input purpose is clearly labeled
- **Error Identification (3.3.1)** — Errors are identified and described in text
- **Error Suggestion (3.3.4)** — Helpful suggestions for correcting errors
- **Focus Visible (2.4.7)** — Clear focus indicator on input fields
