## When to Use

- Selecting a specific time of day (hours and minutes)
- Scheduling appointments or events with precise time selection
- Setting departure, arrival, or meeting times
- Time-dependent form submissions where accuracy matters

## When NOT to Use

- For duration or time intervals — use a duration picker instead
- For timezone-aware times — consider a more specialized component
- For only hour selection without minutes — use a simpler number input
- For very long lists of times — consider a different interaction pattern
