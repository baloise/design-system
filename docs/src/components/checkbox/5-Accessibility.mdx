## WCAG 2.2 AA Compliance

The checkbox component supports patterns that meet WCAG 2.2 Level AA standards:

- **Grouped Inputs (1.3.1)** — Related checkboxes use fieldset/legend
- **Labels (1.3.1)** — Checkboxes have associated labels
- **Focus Visible (2.4.7)** — Clear focus indicator on checkboxes
- **Name, Role, Value (4.1.2)** — Proper semantics and ARIA support
- **Keyboard Accessible (2.1.1)** — Full keyboard navigation support
