## CSS Shadow Parts

Shadow DOM parts are named elements within a web component that can be targeted with CSS selectors from outside the component, allowing you to style specific internal elements without breaking encapsulation.

<br />

**Example:** Target content sections and headers:

```css
/* Style the content container */
ds-content::part(container) {
  max-width: 1200px;
  margin: 0 auto;
}

/* Style section headers */
ds-content::part(header) {
  border-bottom: 1px solid #eee;
  padding-bottom: 16px;
}

/* Style section containers */
ds-content::part(section) {
  margin-bottom: 32px;
}
```

<br />

Primary parts of the content component include:

| Part | Element | Purpose |
| --- | --- | --- |
| `container` | Div | The main content container. |
| `header` | Div | The optional content header area. |
| `section` | Div | Individual content sections. |

## Design Tokens

CSS variables (custom properties) allow you to define reusable values that can be customized globally or per component instance to override default styling without modifying the component code.

<br />

**Example:** Override content styling:

```css
/* Override globally */
:root {
  --ds-content-max-width: 1200px;
  --ds-content-padding: 24px;
}

/* Override for a specific content */
ds-content {
  --content-color-background: #f9f9f9;
}
```
