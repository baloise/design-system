<Lead>
  The **Stack** element is responsible for arranging its direct descendants in a vertical or horizontal direction, while
  also allowing for the inclusion of optional spacing between each child.
</Lead>

<br />
<br />

The **Stack** component deals with layouts in one dimension, whereas the [Grid](?path=/docs/foundation-grid--documentation) manages layouts in two dimensions.
By default, the **Stack** component stacks children vertically in a column.
