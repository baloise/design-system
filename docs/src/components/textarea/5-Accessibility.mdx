## WCAG 2.2 AA Compliance

The textarea component supports patterns that meet WCAG 2.2 Level AA standards:

- **Labels (1.3.1)** — Textareas have properly associated labels
- **Identification of Input (3.3.2)** — Input purpose is clearly labeled
- **Error Identification (3.3.1)** — Validation errors are identified in text
- **Focus Visible (2.4.7)** — Clear focus indicator on textarea fields
- **Text Spacing (1.4.12)** — Text is readable even with adjusted spacing
