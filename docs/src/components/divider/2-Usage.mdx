## When to Use

- Separate different content sections or groups visually
- Break up long pages into manageable sections
- Distinguish between different types of content or data
- Divide layouts into columns or sections
- Provide visual breathing room between content blocks

## When NOT to Use

- Do not use dividers for purely decorative purposes without semantic meaning
- Do not use multiple dividers in close succession — it reduces their visual impact
- For structural separation that should use semantic HTML (e.g., `<section>`, `<article>`)
- In dense data tables where borders would be more appropriate
- As a replacement for proper whitespace and layout — use spacing instead

## Do's and Don'ts

### Placement and Spacing

Use dividers strategically to enhance content organization without cluttering the layout.

### Color Usage

Use divider colors to reinforce meaning or hierarchy. Default grey works for most cases.
