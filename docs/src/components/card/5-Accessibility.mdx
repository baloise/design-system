## WCAG 2.2 AA Compliance

The card component can support patterns that meet WCAG 2.2 Level AA standards:

- **Color Contrast (4.11.1)** — Text and interactive elements meet minimum contrast ratios
- **Keyboard Navigation (2.1.1)** — All interactive card elements are keyboard accessible
- **Focus Visible (2.4.7)** — Clear focus indicators for keyboard users
- **Image Text Alternatives (1.1.1)** — Meaningful alt text for all images
