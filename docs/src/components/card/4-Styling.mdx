## CSS Shadow Parts

Shadow DOM parts are named elements within a web component that can be targeted with CSS selectors from outside the component, allowing you to style specific internal elements without breaking encapsulation.

<br />

**Example:** Target the card container:

```css
/* Style the main card element */
ds-card::part(card) {
  border-radius: 8px;
  background-color: #ffffff;
}
```

<br />

Primary parts of the card component include:

| Part | Element | Purpose |
| --- | --- | --- |
| `card` | Container | The main card wrapper element. |

## Design Tokens

CSS variables (custom properties) allow you to define reusable values that can be customized globally or per component instance to override default styling without modifying the component code.

<br />

**Example:** Override card styling:

```css
/* Override globally */
:root {
  --ds-card-color-background: #ffffff;
  --ds-card-color-border: #e0e0e0;
}

/* Override for a specific card */
ds-card {
  --card-color-background: #f5f5f5;
}
```
