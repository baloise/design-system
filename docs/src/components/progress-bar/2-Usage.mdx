## When to Use

- Display task or operation progress (0–100%)
- Show loading or completion status in workflows
- Indicate multi-step process progress
- Provide visual feedback for long-running operations
- Display file upload or download progress
- Show form completion percentage

## When NOT to Use

- For status indicators that aren't measurable progress — use badge or notification
- For simple loading without progress details — use spinner
- For very large datasets where progress is unreliable — use skeleton loading
- For circular progress animations — use spinner or circular progress component
- For boolean states (done/not done) — use checkbox or toggle

## Do's and Don'ts

### Clear Progress Values

Always provide meaningful progress information. Users need to know how much is complete.
