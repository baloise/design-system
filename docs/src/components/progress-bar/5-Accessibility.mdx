## WCAG 2.2 AA Compliance

The progress-bar component supports patterns that meet WCAG 2.2 Level AA standards:

- **Perceptible (1.3.1)** — ARIA role and live regions convey progress
- **Distinguishable (1.4.3)** — Sufficient contrast for progress fill and background
- **Operable (2.1)** — No keyboard traps or interaction required
- **Understandable (3.3.2)** — Percentage labels explain progress status
- **Robust (4.1.2)** — Proper ARIA roles and attributes
