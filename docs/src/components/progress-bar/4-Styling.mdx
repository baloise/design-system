## CSS Shadow Parts

Shadow DOM parts are named elements within a web component that can be targeted with CSS selectors from outside the component, allowing you to style specific internal elements without breaking encapsulation.

<br />

**Example:** Target the progress bar and label:

```css
/* Style the progress indicator */
ds-progress-bar::part(bar) {
  border-radius: 8px;
}

/* Style the label */
ds-progress-bar::part(label) {
  font-weight: bold;
}
```

<br />

Primary parts of the progress-bar component include:

| Part | Element | Purpose |
| --- | --- | --- |
| `bar` | Div | The progress bar container and fill. |
| `label` | Span | The optional progress label or percentage. |

## Design Tokens

CSS variables (custom properties) allow you to define reusable values that can be customized globally or per component instance to override default styling without modifying the component code.

<br />

**Example:** Override progress-bar styling:

```css
/* Override globally */
:root {
  --ds-progress-bar-color-background: #e0e0e0;
}

/* Override for a specific progress-bar */
ds-progress-bar {
  --progress-bar-color-fill: #4caf50;
}
```
