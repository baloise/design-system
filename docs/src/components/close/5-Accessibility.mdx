## WCAG 2.2 AA Compliance

The close component supports patterns that meet WCAG 2.2 Level AA standards:

- **Target Size (2.5.5)** — Button is at least 44x44 pixels
- **Focus Visible (2.4.7)** — Clear focus indicator on the button
- **Name, Role, Value (4.1.2)** — Proper ARIA label and button role
- **Keyboard Accessible (2.1.1)** — All functionality available via keyboard
