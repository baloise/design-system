## WCAG 2.2 AA Compliance

The logo component supports patterns that meet WCAG 2.2 Level AA standards:

- **Link Purpose (2.4.4)** — Logo link destination is clear from context or label
- **Target Size (2.5.5)** — Logo link is at least 44x44 pixels
- **Focus Visible (2.4.7)** — Logo link has visible focus indicator
- **Motion (2.3.3)** — Animated logo doesn't contain problematic flashing
