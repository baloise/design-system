## When to Use

- **Page layout wrapper** — Constrains main content to a readable max-width on large screens
- **Responsive padding** — Automatically adjusts horizontal padding based on device breakpoints
- **Multiple layout patterns** — Choose between default, fluid, and compact variants depending on content needs
- **Center content** — Uses flex and margin auto to center and vertically fill the viewport

## When NOT to Use

- **Nested deeply** — Avoid stacking multiple containers (use a single wrapper at the top level)
- **Sidebar layouts** — Use grid or custom flex layout for multi-column designs
- **Fixed-width needs** — If you need precise pixel widths, adjust CSS variables directly instead
