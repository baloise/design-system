## WCAG 2.2 AA Compliance

The table component uses native HTML table semantics that meet WCAG 2.2 Level AA standards:

- **1.3.1 Info and Relationships** — `<th>`, `<thead>`, `<tbody>`, and `<tfoot>` elements expose the data structure to assistive technologies without relying on visual presentation.
- **1.4.3 Contrast (Minimum)** — all text and border colours defined via `--ds-table-*` tokens meet the 4.5:1 ratio requirement against their respective background colours.
- **1.4.11 Non-text Contrast** — row borders and the selected-row highlight meet the 3:1 ratio requirement for UI component boundaries.
