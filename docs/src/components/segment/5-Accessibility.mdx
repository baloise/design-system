## WCAG 2.2 AA Compliance

The segment component implements patterns that meet WCAG 2.2 Level AA standards:

- **Keyboard Navigation (2.1.1)** — Arrow keys cycle between options; Tab moves focus in and out of the group
- **Focus Visible (2.4.7)** — A visible focus indicator is shown on the active option
- **Name, Role, Value (4.1.2)** — `role="fieldset"`, `aria-checked`, `aria-invalid`, and `aria-disabled` are kept in sync with the component state
- **Error Identification (3.3.1)** — Invalid state is communicated via `aria-invalid` and a visible `invalid-text` message
- **Labels or Instructions (3.3.2)** — The `label` and `description` props expose a persistent group label and hint text
