<StoryHeading of={SegmentStories.Basic}></StoryHeading>

The default segment renders a horizontal row of options. Set a `value` to pre-select an item, and use `allow-empty-selection` to let users deselect the current choice.

CSS-only equivalent using a `<fieldset>`, `<legend>`, and `<input type="radio">` elements. No JavaScript is required.

<CanvasTabs of={SegmentStories.Basic} htmlOf={SegmentStories.BasicHtml} />

{/* ------------------------------------------------------ */}

<StoryHeading of={SegmentStories.Vertical}></StoryHeading>

Add the `vertical` attribute to stack segment items in a column. Each item can include an optional `description` for additional context.

CSS-only vertical variant using the `is-vertical` modifier class on the segment container.

<CanvasTabs of={SegmentStories.Vertical} htmlOf={SegmentStories.VerticalHtml} />

{/* ------------------------------------------------------ */}

<StoryHeading of={SegmentStories.Disabled}></StoryHeading>

Apply the `disabled` attribute to prevent interaction with all segment items. The selected value is preserved but cannot be changed.

CSS-only disabled variant using the `is-disabled` modifier class and the `disabled` attribute on each `<input>`.

<CanvasTabs of={SegmentStories.Disabled} htmlOf={SegmentStories.DisabledHtml} />

{/* ------------------------------------------------------ */}

{/* ------------------------------------------------------ */}

<StoryHeading of={SegmentStories.Invalid}></StoryHeading>

Apply `invalid` and `invalid-text` to display a validation error message below the group.

CSS-only invalid variant using the `is-danger` modifier class and `aria-invalid="true"` on each `<input>`.

<CanvasTabs of={SegmentStories.Invalid} htmlOf={SegmentStories.InvalidHtml} />

{/* ------------------------------------------------------ */}
{/* ------------------------------------------------------ */}

<StoryHeading of={SegmentStories.Wide}></StoryHeading>

Add the `wide` attribute to make each segment item expand equally and fill the full available width.

<Canvas of={SegmentStories.Wide} />

{/* ------------------------------------------------------ */}

<StoryHeading of={SegmentStories.Icon}></StoryHeading>

Add `icon` to each `ds-segment-item` and `icon-only` to the parent to display icon-only items. Accessible labels are applied automatically.

<Canvas of={SegmentStories.Icon} />
