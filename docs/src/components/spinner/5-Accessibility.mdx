## WCAG 2.2 AA Compliance

The spinner component supports patterns that meet WCAG 2.2 Level AA standards:

- **Status Messages (4.1.3)** — Loading state is announced via ARIA live regions
- **Name, Role, Value (4.1.2)** — Spinner has proper ARIA attributes
- **Color Contrast (4.11.1)** — Spinner is visible against background
- **Animation Concerns (2.3.3)** — No flashing or animations that could trigger seizures
