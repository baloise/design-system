## WCAG 2.2 AA Compliance

The pagination component supports patterns that meet WCAG 2.2 Level AA standards:

- **Semantic Navigation (1.3.1)** — Uses nav element with aria-label
- **Keyboard Accessible (2.1.1)** — Tab navigates between page buttons
- **Focus Visible (2.4.7)** — Active page button clearly marked
- **Link Context (2.4.9)** — Page numbers provide clear context
- **Name, Role, Value (4.1.2)** — Buttons have descriptive labels
