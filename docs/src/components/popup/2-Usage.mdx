## When to Use

- Dropdowns and select menus anchored to a trigger element
- Context menus that appear near the cursor or button
- Autocomplete suggestions or search results
- Floating panels for settings, filters, or quick actions
- Tooltips with rich content or interactive elements
- Popovers that provide additional information or options

## When NOT to Use

- Modal dialogs that block interaction with the page — use `ds-modal` instead
- Drawers that slide from edges — use `ds-drawer` instead
- Simple tooltips with only text — use native title attribute or `ds-tooltip` if available
- Notifications or alerts — use toast or snackbar components
- Content that should always be visible — integrate into the main layout

## Do's and Don'ts

### Trigger Wiring
