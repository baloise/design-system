## WCAG 2.2 AA Compliance

The popup component implements patterns that meet WCAG 2.2 Level AA standards:

- **Dialog Semantics** — The panel uses `role="dialog"` by default, with proper `aria-label` for accessible naming.
- **Focus Management** — When opening, focus moves into the popup. When closing, focus returns to the trigger (if available). Focus trap (Tab navigation) works within the panel.
- **Keyboard Navigation** — Escape key closes the popup. Tab navigates focusable elements. For role="listbox", arrow keys can be used with proper ARIA attributes.
- **Positioned Correctly** — Floating positioning uses `computePosition()` to ensure the popup doesn't get cut off by viewport boundaries.
- **Contrast** — The close button (part="close") and content have sufficient contrast for visibility.
- **Touch-friendly** — Backdrop dismiss provides a clear touch target to close the popup when backdrop-dismiss is enabled.
