## WCAG 2.2 AA Compliance

The accordion component implements patterns that meet WCAG 2.2 Level AA standards:

- **Keyboard Navigation (2.1.1)** — All accordion headers are keyboard accessible
- **Focus Visible (2.4.7)** — Clear focus indicators on interactive elements
- **Name, Role, Value (4.1.2)** — Proper ARIA roles and attributes for state management
- **Heading Structure (1.3.1)** — Maintains semantic heading hierarchy within sections
