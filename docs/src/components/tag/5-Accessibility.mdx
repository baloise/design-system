## WCAG 2.2 AA Compliance

The tag component supports patterns that meet WCAG 2.2 Level AA standards:

- **Color Contrast (4.11.1)** — Text meets minimum contrast ratios against background
- **Color Not Only Means (1.4.1)** — Color is not the only way to convey information
- **Focus Visible (2.4.7)** — Close buttons have visible focus indicators
- **Name, Role, Value (4.1.2)** — Proper semantic HTML and ARIA attributes
