## Accessibility

- Always include visible text content (a number, abbreviation, or label) inside the badge so screen readers can announce it.
- When using the `icon` prop, add an `aria-label` to the **parent element** that describes what the badge indicates — for example `<button aria-label="Notifications, 3 unread">`.
- Do not rely on color alone to convey meaning; pair color variants with surrounding text or semantic context.
- For purely decorative badges, add `aria-hidden="true"` to the `<ds-badge>` element.
