## When to Use

- Display notification counts (e.g., unread messages, pending items)
- Show status information (e.g., online/offline, active/inactive)
- Highlight new or updated content
- Indicate priority levels or importance

## When NOT to Use

- For displaying detailed information (use a tooltip or popover instead)
- As a standalone component (badges should always be associated with another component)
- For large amounts of text (badges should contain short, concise content)

## Do's and Don'ts

### Text Content Examples

The most important guideline is keeping badge content short and concise. Here's how to do it right:
