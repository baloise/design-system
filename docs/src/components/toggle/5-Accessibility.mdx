## WCAG 2.2 AA Compliance

The toggle component supports patterns that meet WCAG 2.2 Level AA standards:

- **Semantics (1.3.1)** — Uses role="switch" and aria-checked
- **Color Contrast (1.4.3)** — Sufficient contrast between on/off states
- **Keyboard Navigation (2.1.1)** — Space/Enter toggle the switch
- **Focus Visible (2.4.7)** — Clear focus indicator
- **Name, Role, Value (4.1.2)** — ARIA attributes properly identify switch
