## When to Use

- Add decorative visual accents to layouts
- Create abstract geometric designs
- Add subtle background shapes to sections
- Enhance visual hierarchy without content
- Provide design system branded shapes
- Create visual separation between sections

## When NOT to Use

- For meaningful content — use actual content elements
- For interactive elements — shapes are purely decorative
- For SEO-critical content — shapes don't contribute to page meaning
- For functional icons — use the icon component
- For complex illustrations — use custom SVG or images
- For accessible diagrams — shapes may not convey meaning to screen readers
