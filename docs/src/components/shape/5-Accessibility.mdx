## WCAG 2.2 AA Compliance

The shape component supports patterns that meet WCAG 2.2 Level AA standards:

- **Semantics (1.3.1)** — Decorative shapes use aria-hidden
- **Color Contrast (1.4.3)** — Sufficient contrast for visible shapes
- **Meaningful Content (1.4.11)** — Shapes don't obscure important content
- **Text Alternatives (1.1.1)** — Meaningful shapes have aria-label or alt text
