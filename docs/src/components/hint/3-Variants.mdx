{/* STORIES */}
{/* ------------------------------------------------------ */}

<StoryHeading of={HintStories.Hint}></StoryHeading>

The basic hint with title and text using `ds-hint-title` and `ds-hint-text` sub-components. On desktop it opens a popup to the right; on touch viewports it opens a drawer from the bottom.

<CanvasTabs of={HintStories.Hint} />

{/* ------------------------------------------------------ */}

<StoryHeading of={HintStories.WithSlots}></StoryHeading>

Hint content can be provided using raw HTML slots instead of sub-components. Use the `title` slot for the heading and the default slot for body content.

<CanvasTabs of={HintStories.WithSlots} />

{/* ------------------------------------------------------ */}

<StoryHeading of={HintStories.CustomTriggerLabel}></StoryHeading>

Customize the accessible label on the trigger button with the `triggerLabel` prop. This is important for screen readers and provides context.

<CanvasTabs of={HintStories.CustomTriggerLabel} />

{/* ------------------------------------------------------ */}

<StoryHeading of={HintStories.PlacementTop}></StoryHeading>

Control where the popup opens on desktop using the `placement` prop. This variant opens above the trigger.

<CanvasTabs of={HintStories.PlacementTop} />

{/* ------------------------------------------------------ */}

<StoryHeading of={HintStories.PlacementBottom}></StoryHeading>

The popup can open below the trigger. Useful when there's more space below the element.

<CanvasTabs of={HintStories.PlacementBottom} />

{/* ------------------------------------------------------ */}

<StoryHeading of={HintStories.PlacementLeft}></StoryHeading>

The popup can open to the left of the trigger. Useful in right-aligned layouts or narrow columns.

<CanvasTabs of={HintStories.PlacementLeft} />

{/* ------------------------------------------------------ */}

<StoryHeading of={HintStories.CustomCloseLabel}></StoryHeading>

On touch viewports, the drawer shows a close button. Customize its label with the `closeLabel` prop. When omitted, the label is localized based on the app language configuration.

<CanvasTabs of={HintStories.CustomCloseLabel} />

{/* ------------------------------------------------------ */}

<StoryHeading of={HintStories.NoTitle}></StoryHeading>

A hint can contain just body text without a title. Useful for brief explanations that don't need a heading.

<CanvasTabs of={HintStories.NoTitle} />

{/* ------------------------------------------------------ */}

<StoryHeading of={HintStories.Inline}></StoryHeading>

Hints work well inline within sentences or paragraphs, providing context right where it's needed without breaking the flow.

<CanvasTabs of={HintStories.Inline} />
