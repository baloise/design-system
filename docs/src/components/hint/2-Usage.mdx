## When to Use

- Provide additional context or explanation for form fields or UI elements
- Offer tips or guidance without disrupting the main workflow
- Inline help within sentences or paragraphs
- Contextual information that's optional but valuable

## When NOT to Use

- Critical instructions that users must read — use persistent text or tooltips
- Essential validation messages — place these inline with the field
- Long explanations that require scrolling — use a separate help page
- Information that should always be visible — avoid hiding behind a trigger

## Do's and Don'ts

### Clear and Concise Content

### Contextual Placement

        </div>
      ),
      description: 'Avoid hints that are separated from their context.',
    },
  ]}
