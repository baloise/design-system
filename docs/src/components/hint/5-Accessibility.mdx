## WCAG 2.2 AA Compliance

The hint component implements patterns that meet WCAG 2.2 Level AA standards:

- **Button Semantics** — The trigger is a semantic button with `aria-label` and `aria-expanded` to communicate its state and purpose.
- **Dialog Role** — The hint panel has `role="dialog"` on desktop (popup) and uses the native dialog semantics on touch (drawer).
- **Focus Management** — Focus is managed automatically: when the hint opens, focus moves into the panel; when closed, focus returns to the trigger.
- **Keyboard Navigation** — Users can open/close via click or Enter/Space. Escape closes the panel on both desktop and mobile.
- **Screen Reader Announcements** — The trigger label and panel role are announced, allowing screen reader users to understand the hint's purpose and content.
- **Icon with Label** — The info-circle icon is marked as `aria-hidden` since the button label provides sufficient context.
