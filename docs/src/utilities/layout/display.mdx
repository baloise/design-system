## Hidden

An element that is hidden is not visible and does not occupy any space.

<CanvasWithCodePen of={DisplayStories.Hidden} />

## Block

A block element begins on a new line.

<CanvasWithCodePen of={DisplayStories.Block} />

## Inline

An inline element does not initiate on a new line and takes up width as required.

<CanvasWithCodePen of={DisplayStories.Inline} />

## Inline Block

Inline block behaves much like an inline element, but it also respects properties such as width, height, and top/bottom paddings/margins.

<CanvasWithCodePen of={DisplayStories.InlineBlock} />

## Flex

Sets the element to display as a block-level flex container.

<CanvasWithCodePen of={DisplayStories.Flex} />

## Inline Flex

Sets the element to display as an inline-level flex container.

<CanvasWithCodePen of={DisplayStories.InlineFlex} />

## Responsive

Tailored responsive options are at your disposal for customizations according to screen size.
Simply append a responsive breakpoint keyword with a dash, like `tablet:hidden`, to apply the corresponding responsive class.
