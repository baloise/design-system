## Form fields

A basic form field has his container element `ds-field` and 3 child elements:

- `ds-field-label` for adding a label to the form control.
- `ds-field-control` to contain the control like an input.
- `ds-field-message` to show validation message or helper messages.

<Code
  language="html"
  code={`
<ds-field>
  <ds-field-label>Firstname</ds-field-label>
  <ds-field-control>
    <ds-input placeholder="Enter your firstname"></ds-input>
  </ds-field-control>
  <ds-field-message color="danger">Required Field</ds-field-message>
</ds-field>
`}
/>

## Form Grid

To combine form fields / controls to a bigger form use the [Form Grid component](?path=/docs/components-form-form-grid--basic).

<Code
  language="html"
  code={`

      <ds-field-control>

## Reactive Forms

The form elements support [Angular Reactive Forms](https://angular.io/guide/reactive-forms).
Below is a basic example to use the reactive form together with the Design System.

<InfoQuote>

**Tip**

Check out the repository [baloise/web-app-utils](https://github.com/baloise/web-app-utils/blob/master/packages/validators/README.md) for Baloise
specific validators.

</InfoQuote>

1. Angular package offers `balAutoFocus` directive which brings focus to the element.
   Usage can be seen in the following code in ds-field.
   In order for it to work please import `BalSharedModule` in each Angular module that uses it.

<Code
  language="html"
  noPreview
  code={`
<form [formGroup]="form" (ngSubmit)="onSubmit()" class="grid is-multiline mt-none">
  <ds-field class="col is-full py-none" required>
    <ds-field-label required>Email</ds-field-label>
    <ds-field-control>
      <ds-input name="email" placeholder="Enter your email" formControlName="email" balAutoFocus></ds-input>
    </ds-field-control>
    <ds-field-message>
      <ds-ng-error controlName="email" error="isRequired">This field is required</ds-ng-error>
      <ds-ng-error controlName="email" error="isMinLength">Min length is 4</ds-ng-error>
      <ds-ng-error controlName="email" error="isEmail">Not a valid email</ds-ng-error>
    </ds-field-message>
  </ds-field>
  <ds-field class="col is-half py-none">...</ds-field>
</form>
`}
/>

2. In the component class wen can define the validators for the form control.

<Code
  language="ts"
  code={`
import { Component } from '@angular/core'
import { FormControl, FormGroup, Validators } from '@angular/forms'

@Component({
${' '}${' '}selector: 'app-form-page',

})
export class FormPageComponent {
${' '}${' '}form = new FormGroup({
${' '}${' '}${' '}${' '}email: new FormControl(null, [Validators.required]),
${' '}${' '}})

${' '}${' '}onSubmit() {
${' '}${' '}${' '}${' '}alert('Form is submitted!')
${' '}${' '}}
}
`}
/>
