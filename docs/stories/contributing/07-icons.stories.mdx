## Add new icons

### UI Icons

1. Download the svg icon 16x16 from https://www.streamlinehq.com/icons/core-solid.
2. Place it at packages/icons/src/assets.

### Brand Icons

1. Download the svg icons in all colors from https://brand.baloise.com/.
2. Place it at packages/brand-icons/src/assets.

## Build the package

To optimize the svg icons for web components run this command:

```bash
npm run optimize
```
