## Start

Elements are positioned at the beginning of the container.

<Code
  language="html"
  code={`
<div class="flex align-items-start flex-wrap gap-normal" style="min-height: 200px">
  <div class="bg-green-3 radius-normal flex justify-content-center align-items-center" style="min-width: 200px; min-height: 50px">1</div>
  <div class="bg-green-3 radius-normal flex justify-content-center align-items-center" style="min-width: 200px; min-height: 150px">2</div>
  <div class="bg-green-3 radius-normal flex justify-content-center align-items-center" style="min-width: 200px; min-height: 100px">3</div>
</div>
`}
/>
