## UI Icons

A minimal set of UI-Icons is already available from the beginning. Here are some [more UI-Icons](?path=/docs/foundation-icons--documentation#overview), which can be imported

<Canvas of={IconStories.UiIcons} sourceState="none" />

To incorporate icons beyond this basic set, you can either **import them during initialization** or **import the raw SVG icon** and pass it to the component using the `svg` property.

<AngularFramework>

### Import during initialization

Import the icons form the `@baloise/design-system-icons` package and pass them with the BalConfig during the initialization.

```ts
import { balIconStarFull } from '@baloise/design-system-icons'

@NgModule({
  imports: [
    BaloiseDesignSystemModule.forRoot({
      defaults: {
        icons: { balIconStarFull },
      },
    }),
  ],
})
export class AppModule {}
```

### Import into the component

When importing an SVG into your component you decrease the initial page load. So you use the SVG only where you needed and at the time you wanted.

```ts
import { balIconStarFull } from '@baloise/design-system-icons'

@Component({...})
export class AppComponent {
  myIcon = balIconStarFull
}

```

```html
<bal-icon [svg]="myIcon"></bal-icon>
```

</AngularFramework>

<ReactFramework>

### Import during initialization

Import the icons form the `@baloise/design-system-icons` package and pass them with the BalConfig during the initialization.

```ts
import React from 'react'
import { useBaloiseDesignSystem, BalApp } from '@baloise/design-system-components-react'
import { balIconStarFull } from '@baloise/design-system-icons'

export default function App() {
  useBaloiseDesignSystem({
    defaults: {
      icons: { balIconStarFull },
    },
  })

  return <BalApp>...</BalApp>
}
```

### Import into the component

When importing an SVG into your component you decrease the initial page load. So you use the SVG only where you needed and at the time you wanted.

```ts
import React from 'react'
import { BalIcon } from '@baloise/design-system-components-react'
import { balIconStarFull } from '@baloise/design-system-icons'

export default function Example() {
  return <BalIcon svg={balIconStarFull}></BalIcon>
}
```

</ReactFramework>

<VueFramework>

### Import during initialization

Import the icons form the `@baloise/design-system-icons` package and pass them with the BalConfig during the initialization.

```ts
import { balIconStarFull } from '@baloise/design-system-icons'

createApp(App)
  .use(
    createBaloiseDesignSystem({
      defaults: {
        icons: { balIconStarFull },
      },
    }),
  )
  .mount('#app')
```

### Import into the component

When importing an SVG into your component you decrease the initial page load. So you use the SVG only where you needed and at the time you wanted.

```html
<script lang="ts" setup>
  import { BalIcon } from '@baloise/design-system-components-vue'
  import { balIconStarFull } from '@baloise/design-system-icons'
</script>

<template>
  <BalIcon :svg="balIconStarFull"></BalIcon>
</template>
```

</VueFramework>

<HTMLFramework>

### Import during initialization

Import the icons form the `@baloise/design-system-icons` package and pass them with the BalConfig during the initialization.

```ts
import { initialize } from '@baloise/design-system-components'
import { balIconStarFull } from '@baloise/design-system-icons'

initialize({
  icons: { balIconStarFull },
})
```

### Import into the component

When importing an SVG into your component you decrease the initial page load. So you use the SVG only where you needed and at the time you wanted.

```js
import { balIconStarFull } from '@baloise/design-system-icons'

document.getElementById('my-icon').svg = balIconStarFull
```

</HTMLFramework>

<StoryHeading of={IconStories.BrandIcons}></StoryHeading>

Brand icons or custom SVG images can be added with the property `svg` and set color to `auto` to use the colors of the imported SVG image.

<Canvas of={IconStories.BrandIcons} sourceState="none" />

Install the `@baloise/design-system-brand-icons` to the project to be able to access the svg file.

```bash
npm add @baloise/design-system-brand-icons
```

<AngularFramework>

```ts
import { balBrandIconCarCrashWithAnimalGreen } from '@baloise/design-system-brand-icons'

@Component({...})
export class AppComponent {
  myIcon = balBrandIconCarCrashWithAnimalGreen
}

```

```html
<bal-icon color="auto" [svg]="myIcon"></bal-icon>
```

</AngularFramework>

<ReactFramework>

```ts
import React from 'react'
import { BalIcon } from '@baloise/design-system-components-react'
import { balBrandIconCarCrashWithAnimalGreen } from '@baloise/design-system-brand-icons'

export default function Example() {
  return <BalIcon svg={balBrandIconCarCrashWithAnimalGreen}></BalIcon>
}
```

</ReactFramework>

<VueFramework>

```html
<script lang="ts" setup>
  import { BalIcon } from '@baloise/design-system-components-vue'
  import { balBrandIconCarCrashWithAnimalGreen } from '@baloise/design-system-brand-icons'
</script>

<template>
  <BalIcon :svg="balBrandIconCarCrashWithAnimalGreen"></BalIcon>
</template>
```

</VueFramework>

<HTMLFramework>

```js
import { balBrandIconCarCrashWithAnimalGreen } from '@baloise/design-system-brand-icons'

document.getElementById('my-icon').svg = balBrandIconCarCrashWithAnimalGreen
```

</HTMLFramework>
