Ther is a new way of adding a options to the bal-radio-group
by introducing the `options` property in which options can be passed using a Javascript.

```html
<bal-radio-group [options]="options"></bal-radio-group>
```

```typescript
import { newBalRadioOption } from '@baloise/design-system-components'

@Component({
  selector: 'app-example-component',
  templateUrl: './example-component.component.html',
})
export class ExampleComponent {
  options = [
    newBalRadioOption({ label: 'Label 1', value: '1' }),
    newBalRadioOption({
      label: () => 'Label with <a class="is-link" href="http://baloise.com">Link</a>',
      value: '2',
    }),
  ]
}
```

The radio buttons are have a built in grid system. Just add the props `grid`, `grid-mobile` or `grid-tablet` to the `bal-radio-group`.

On mobile devices, the default setting allows for one item per line. To override this behavior, utilize the `grid-mobile` prop.
