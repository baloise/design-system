<Lead>
  **Input** component is a wrapper to the HTML input element with custom styling and additional functionality. It
  accepts most of the same properties as the HTML input, but works great on desktop devices and integrates with the
  keyboard on mobile devices. It is meant for text type inputs only, such as "text", "password", "email", "number",
  "search", "tel", and "url". It supports all standard text input events including keyup, keydown, keypress, and more.
</Lead>

<StoryHeading of={InputStories.InputDate}></StoryHeading>

This is a dedicated component for date inputs. It employs a mask to ensure the entry of dates in the correct format (dd.mm.yyyy).
The format adapts automatically based on the locale of the Design System. Separation characters, such as the dot (.),
facilitate seamless navigation between individual blocks of the date. Additionally, the entered date is automatically
formatted upon leaving the input field.
