## Typeface

Our custom-made corporate typeface is called Baloise Create. It is a typeface with a solid geometric character and a large x-height supporting todayâ€™s digital needs. The typeface is available in two families: Baloise Create Headline, which is used in titles and bold leads. Baloise Create text is the typeface for large amounts of text. It is optimized for the best readability in print as well as in digital.

### Baloise Create Families

Baloise Create Headline and Baloise Create Text. Each family combines the four weights, Light, Regular and Bold.

#### Baloise Create Headline

Baloise Create Headline combines elementary shapes and precise construction. It's shaped by pure forms, low stroke modulation and almost perfect circles. It's a typeface of quirks and clarity.

#### Baloise Create Text

Where Baloise Create Headline revels in pure geometry, Baloise Create Text has a practical focus. With a robust x-height, open apertures and generous spacing. An all-rounder specialized in smaller reading sizes and lower resolutions, suitable for user interfaces and long-form writing. Diverse in language and clear screen performance.
