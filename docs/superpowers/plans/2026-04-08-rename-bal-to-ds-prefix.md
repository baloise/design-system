# Rename Web Component Prefix `bal-` → `ds-` Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the `bal-` HTML custom element prefix with `ds-` across all Stencil component tag declarations, render trees, SCSS selectors, test fixtures, Playwright files, and Storybook documentation.

**Architecture:** This is a bulk rename — no new logic is added. Each task targets a specific file group and applies sed replacements followed by grep verification to confirm zero remaining occurrences. Auto-generated files (`components.d.ts`, `api.md`) are intentionally skipped; they regenerate correctly on the next build once the source tags are renamed.

**Tech Stack:** Stencil.js (web components), SCSS, Playwright, Storybook (HTML + MDX stories), Nx monorepo, macOS sed (`sed -i ''`)
